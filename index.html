Proyectos Académicos y Profesionales

Horario Académico - Unidad Carlos Pomerio Zambrano

Un programa informático diseñado específicamente para este propósito puede ser una herramienta necesaria para la Unidad Educativa Carlos Pomerio Zambrano, optimizando la organización y el manejo de horarios para todas las materias de la institución.

Funcionalidades:

  • Accede a tu horario completo en diferentes vistas: semanal, mensual.
  • Edición de horario para cualquier caso
  • Recibe alertas sobre próximos exámenes y otras fechas importantes.
  • Filtro para la consulta del horario por curso o materia

Especificaciones:

El sistema está desarrollado en PHP, JavaScript y MySQL.
