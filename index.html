Proyectos Académicos y Profesionales

Horario Académico - Unidad Carlos Pomerio Zambrano

Para crear un sistema de horario efectivo, primero identifiqué los requisitos clave, como la autenticación de usuarios, la gestión y visualización de horarios, y las notificaciones automáticas. Implementé una interfaz intuitiva de usuario con páginas de inicio de sesión, visualización de horarios organizada y formularios de creación de horarios simples.

Funcionalidades:

  • Accede a tu horario completo en diferentes vistas: semanal, mensual.
  • Edición de horario para cualquier caso
  • Recibe alertas sobre próximos exámenes y otras fechas importantes.
  • Filtro para la consulta del horario por curso o materia

Comercial Jennifer Sistema de Facturacion

El proyecto "Comercial Jennifer - Programa de Punto de Venta" consistió en desarrollar e implementar un sistema integral de punto de venta para la gestión eficiente de inventario, facturación y ventas. Utilizando tecnologías como PHP, JavaScript y MySQL, se logró crear una plataforma robusta y escalable. Las principales funcionalidades incluyeron el registro de productos y clientes, gestión de stock, generación de facturas electrónicas, reportes de ventas y análisis de rendimiento.

Funcionalidades:

  • Punto de venta.
  • Libro Diario
  • Kardex.
  • Facturas
